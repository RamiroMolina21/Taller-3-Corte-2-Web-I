Gestión de Estudiantes Gestión de Programas Academicos Gestión de Matriculas

El sistema debe permitir gestionar la información de los estudiantes inscritos. Se requiere almacenar los siguientes datos para cada estudiante:

  • Identificación del estudiante (Número único alfanumérico) - Obligatorio y único para cada estudiante.
  • Nombre completo - Obligatorio y con validación de caracteres alfabéticos únicamente.
  • Fecha de nacimiento - Obligatorio y debe ser válida (no permite fechas futuras).
  • Correo electrónico - Obligatorio y debe cumplir con el formato de correo electrónico (Ejemplo: ejemplo@dominio.com).
  • Teléfono - Opcional, pero si se ingresa, debe ser numérico y tener entre 8 y 15 dígitos.

El sistema debe permitir gestionar los programas académicos disponibles para los estudiantes. Se requiere almacenar la siguiente información para cada programa:

  • Código del programa (alfanumérico único) - Obligatorio y único para cada programa.
  • Nombre del programa - Obligatorio y no puede repetirse en la lista de programas.
  • Duración (en semestres) - Obligatorio y debe ser un valor numérico positivo.
  • Modalidad (presencial, virtual, semipresencial) - Obligatorio y debe seleccionarse de una lista predefinida de opciones.
  • Fecha de inicio - Opcional, pero si se ingresa, debe ser una fecha pasada o actual.

El sistema debe permitir registrar la matrícula de estudiantes en los programas académicos. Cada matrícula debe asociar a un estudiante con un programa, incluyendo los siguientes datos:

  • Identificación de matrícula (Número único alfanumérico) - Obligatorio y único para cada matrícula.
  • Identificación del estudiante - Obligatorio y debe corresponder a un estudiante registrado en el sistema.
  • Código del programa - Obligatorio y debe corresponder a un programa registrado en el sistema.
  • Fecha de matrícula - Obligatorio y debe ser una fecha actual o futura.
  • Estado de la matrícula (activa, en proceso, finalizada, cancelada) - Obligatorio y debe seleccionarse de una lista predefinida.
